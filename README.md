 ## alu-AirBnB_clone

## Description
This is the first step in building a full web application: an AirBnB clone. 
This project implements a command interpreter to manage AirBnB objects.

## Command Interpreter

### How to Start
```bash
./console.py
```

### How to Use
The console supports the following commands:
- `create <class>` - Creates a new instance
- `show <class> <id>` - Shows an instance
- `destroy <class> <id>` - Deletes an instance
- `all [<class>]` - Shows all instances or instances of a class
- `update <class> <id> <attribute> <value>` - Updates an instance
- `quit` or `EOF` - Exits the console
